🌱 Smart Hydroponics App

A cross-platform mobile app built using Expo for monitoring and controlling a smart hydroponics system.

⸻

🚀 Setup Instructions

1. Install Node.js

Download and install the LTS version from:
https://nodejs.org

⸻

2. Clone the repository

git clone https://github.com/MadhavBaghla2004/Hydroponics-App.git
cd Hydroponics-App

⸻

3. Install dependencies

npm install

⸻

4. Install Expo Go (on your phone)

* iPhone → App Store
* Android → Play Store
    Search for Expo Go

⸻

5. Run the app

npx expo start --tunnel

⸻

6. Open the app

* Open Expo Go
* Scan the QR code
* The app will load on your phone

⸻

🔁 Updating the app

After pulling new changes:

git pull
npx expo start --tunnel

⸻

⚠️ Notes

* Keep the terminal running while using the app
* Press r in terminal to reload
* Use --clear if something breaks:

npx expo start --tunnel --clear

⸻

🛠 Tech Stack

* React Native (Expo)
* JavaScript / TypeScript
